# Проекторний Годинник (FullClock)

Проекторний Годинник — це веб-застосунок на основі React та Vite, призначений для відображення годинника на весь екран з можливістю проектування на додатковий дисплей.

## Особливості

- **Проекторний режим**: Можливість відкрити годинник на другому екрані (проекторі) з використанням Window Management API.
- **Повноекранний режим**: Зручний перегляд на весь екран.
- **Налаштування відображення**: Можливість сховати/показати секунди.
- **Масштабування**: Налаштування розміру шрифту для часу та дати окремо.
- **Синхронізація**: Налаштування автоматично зберігаються в `localStorage` і синхронізуються між основним вікном та вікном проектора.

## Технології

- [React](https://react.dev/) 19
- [Vite](https://vitejs.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- TailwindCSS

## Встановлення та запуск

1. Переконайтеся, що у вас встановлений Node.js.
2. Встановіть залежності:
   ```bash
   npm install
   ```
3. Запустіть сервер розробки:
   ```bash
   npm run dev
   ```
4. Відкрийте http://localhost:5173 у вашому браузері.

## Скрипти

- `npm run dev` - запуск локального сервера розробки
- `npm run build` - збірка проекту для продакшену
- `npm run preview` - попередній перегляд зібраного проекту

## Використання

- **Проектор**: Натисніть іконку проектора, щоб відкрити годинник на додатковому дисплеї (потребує дозволу Window Management API).
- **Секунди**: Натисніть іконку годинника, щоб увімкнути/вимкнути відображення секунд.
- **Налаштування**: Натисніть іконку налаштувань для зміни масштабу часу та дати.
- **Повний екран**: Натисніть іконку розширення в правому нижньому кутку або клавішу `F11`.
